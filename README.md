# ?? SoleStore — Shoe E-Commerce Website

A full-stack shoe e-commerce website built with React, Django REST Framework, and PostgreSQL.

## ?? Features
- 20 fully working pages
- Product catalog with filters and pagination
- Shopping cart and wishlist
- User authentication (login/register)
- Checkout flow
- Blog with full articles
- Fully responsive design

## ??? Tech Stack
- **Frontend:** React, Zustand, React Router, React Icons
- **Backend:** Django, Django REST Framework
- **Database:** PostgreSQL

## ?? How to Run

### Backend
cd backend
python manage.py runserver 8001

### Frontend
cd frontend
npm start

Open http://localhost:3000
